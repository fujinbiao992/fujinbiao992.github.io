Vue-watch-computed的区别

methos与watch和computed都是以函数为基础的

1- computed

####需要注意的是,就算在data中没有直接声明出要计算的属性,也可以直接在computed中写入,计算属性默认的只有getter属性,在需要的时候写上setter

在控制台输入直接运行vm.fullName = ‘bibi wang’，相应的firstName和lastName也会改变。

2-watch

watch和computed两者很相似,watch用于观察和监听页面上的vue实例,在大部分情况下会使用computed,但是如果在数据变化的同时进行异步操作或者是比较大的开销,那么watch为最佳的选择,watch为一个对象,键是需要观察的表达式,值是对应的回调函数,值也可以是方法名,或者包含选项的对象
